## Cos'è

Anima è un'applicazione desktop Python per creare e gestire personaggi SillyTavern senza conoscere la sintassi STScript. Un wizard guidato installa automaticamente la character card, il sistema Quick Reply con variabili di scena, l'Author's Note e il Lorebook (statico/dinamico) — tutto con un doppio click.

## Prerequisiti

- SillyTavern installato e funzionante
- Python 3.10+
- `pip install customtkinter`

## Come si usa

1. Scarica `anima.pyw`
2. Esegui `pip install customtkinter`
3. Doppio click su `anima.pyw`
