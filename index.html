// cos'è

Costruito per chi usa SillyTavern ogni giorno.

Anima è una piccola app Python che si affianca a SillyTavern e ti permette di creare character card complete attraverso un wizard guidato — nessun JSON da aprire, nessun file PNG da modificare a mano.

Genera automaticamente i Quick Reply set, le Author's Notes e il sistema di variabili di sessione che tiene traccia di umore, ospiti e storia della partita.

Pensato da un utente ST per utenti ST.

// requisiti

  • SillyTavern installato
  • Python 3.10+
  • CustomTkinter (pip install)
  • Windows 10/11

// funzionalità

Tutto quello che ti serve per creare un personaggio completo.

🧙

Wizard guidato

Crea un personaggio passo dopo passo: nome, descrizione, scenario, personalità, saluti alternativi. Nessun campo da indovinare.

⚡

Quick Reply automatici

Genera i set di Quick Reply con i pulsanti corretti — Tempo, Ospiti, Umore, Nota, Storia, Aggiorna, Inietta — già configurati e pronti.

📝

Author's Note

Costruisce e inietta l'Author's Note che riassume le variabili di sessione attive. Il modello sa sempre dove si trova la storia.

🗂

Dashboard personaggi

Visualizza tutti i personaggi installati, modifica le card esistenti e gestisci lorebook e note da un unico pannello.
